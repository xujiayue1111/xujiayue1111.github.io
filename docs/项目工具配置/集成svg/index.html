SVG

在开发项目的时候经常会用到 svg 矢量图，而且我们使用 SVG 以后，页面上加载的不再是图片资源。

这对页面性能来说是个很大的提升，而且SVG 文件比 img 要小的很多，放在项目中几乎不占用资源。

安装 SVG 依赖插件​

pnpm install vite-plugin-svg-icons -D

在vite.config.ts中配置插件​

这里 plugin 数组里如果已经有 vue()，可以在 vue()后面加个逗号，然后将 createSvgIconPlugin 放进去

import { createSvgIconsPlugin } from 'vite-plugin-svg-icons'
import path from 'path'
export default () => {
  return {
    plugins: [
      createSvgIconsPlugin({
        // 将来的svg矢量图标是放在src下的assets的icons文件夹下面
        iconDirs: [path.resolve(process.cwd(), 'src/assets/icons')],
        // Specify symbolId format
        symbolId: 'icon-[dir]-[name]',
      }),
    ],
  }
}

入口文件导入​

// svg插件需要的配置代码
import 'virtual:svg-icons-register'

目标图标导入​

1.在阿里图标库里找到目标图标然后复制 svg 代码​

2.在 assets 目录下创建 SvgIcon 文件然后按需求创建一个个的 svg 文件​

3.将复制的 svg 代码一个个放入 svg 文件之中​

svg 的使用​

在指定的组件下借助<svg>和<use>标签进行使用

<!-- 测试svg图标 -->
<!-- svg:图标外层的容器节点 内部需要与use标签结合使用 -->
<!-- key value一致省略value -->
<svg style="width:30px;height:30px">
    <!-- xlink:href表示要用哪个图标 属性值务必是#icon-svg图片名字 -->
    <!-- use标签的fill属性可以设置图标的颜色 不过有的图标可以设置颜色另一些可能就设置不了 -->
    <!-- 图标的大小也是可以调整的 通过style就可以 -->
    <use xlink:href="#icon-home" fill="red"></use>
</svg>

如果项目比较大，还是将 svg 封装成组件且最好是全局组件，不需要引入直接使用，这样调用起来比较方便，不用在每个需要的地方具体的写一些代码。

在SvgIcon组件中写

//template中
<!-- 测试svg图标 -->
<!-- svg:图标外层的容器节点 内部需要与use标签结合使用 -->
<!-- key value一致省略value -->
<svg :style="{ width, height }">
    <!-- use表示要用哪个图标 属性值务必是#icon-svg图片名字 -->
    <!-- use标签的fill属性可以设置图标的颜色 不过有的图标可以设置颜色另一些可能就设置不了 -->
    <!-- 图标的大小也是可以调整的 通过style就可以 -->
    <use :xlink:href="prefix + name" :fill="color"></use>
</svg>
//script中
//   用来接收别的组件传递的参数
defineProps({
    // 定义xlink:href属性的前缀
    prefix: {
        type: String,
        default: '#icon-'
    },
    // 提供使用的图标的name
    name: String,
    // 接收父组件传递的颜色
    color: {
        type: String,
        // 默认是空 自己本来的颜色
        default: ''
    },
    // 接收父组件传递过来的宽度
    width: {
        type: String,
        default: '16px'
    },
    // 接收父组件传递过来的高度
    height: {
        type: String,
        default: '16px'
    }
})

在其他引入svg的组件中写

//template中
<SvgIcon name="home" color="pink" width="100px" height="100px"></SvgIcon>
//script中
import SvgIcon from '@/components/SvgIcon/index.vue'

key 和 value 一致，省略 value。（k,v 一致省略 v）

在这里还可以在网页中访问svg图片，比如说我把一个图标放在了assets/icons/vip.svg中，我就可以复制这个文件路径然后在网页中输入这个路径进行访问svg图标。

file:///D:/%E5%89%8D%E7%AB%AF%E5%AD%A6%E4%B9%A0/%E9%A1%B9%E7%9B%AE/vue3_admin_template/project/src/assets/icons/vip.svg
