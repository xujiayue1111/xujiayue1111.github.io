sass

如果要使用sass，项目要安装sass-loader，但是需要加上lang="scss"，不然sass语法不会生效。

<style scoped lang="scss"></style>

接下来我们为项目添加一些全局的样式

在src/styles目录下创建一个index.scss文件，当然项目中需要用到清除默认样式，因此在index.scss引入reset.scss，这个reset.scss也是要自行创建在styles目录下。里面的代码可以使用现成的，比如说可以在npm官网去搜索reset.scss，复制里面code的内容就好。

@import reset.scss

在入口文件main.ts引入

import '@/styles/index.scss'

但是在src/styles/index.scss全局样式文件中没有办法使用$变量。因此需要给项目中引入全局变量$.

在style文件夹下创建一个variable.scss文件，这里variable不是固定叫这个名字叫别的名字也可以。

在vite.config.ts文件配置如下:

export default defineConfig((config) => {
    css: {
      preprocessorOptions: {
        scss: {
          javascriptEnabled: true,
          additionalData: '@import "./src/styles/variable.scss";',
        },
      },
    },
    }
}

具体是要如下配置，css只是配置对象里的一个。

export default defineConfig({
    plugins: [vue(),
    createSvgIconsPlugin({
        // 将来的svg矢量图标是放在src下的assets的icons文件夹下面
        iconDirs: [path.resolve(process.cwd(), 'src/assets/icons')],
        // Specify symbolId format
        symbolId: 'icon-[dir]-[name]',
    })],
    resolve: {
        alias: {
            "@": path.resolve("./src") // 相对路径别名配置，使用 @ 代替 src
        }
    },
    // scss全局变量的配置
    css: {
        preprocessorOptions: {
          scss: {
            javascriptEnabled: true,
            // 所有的全局变量都放到了下面这个文件之中
            additionalData: '@import "./src/styles/variable.scss";',
          },
        },
      }
})

这样配置完毕之后，在variable文件里写入任何样式变量都可以在组件中直接使用。

在variable.scss文件中

$color: red;

在对应组件中

div{
    h1{
        color: $color;
    }
}
