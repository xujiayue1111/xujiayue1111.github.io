修改CSS选择器的优先级

1.选择器进行叠加嵌套​

如果想要覆盖原有选择器的样式，可以选择写一个带相同选择器的样式，添加到原有样式的后面。

2.给选择器添加!important，如果已经有一个!important了，可以在这个基础上添加优先级很高的选择器，比如说id选择器。对于内联样式的覆盖也是选择添加!important。​
